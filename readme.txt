A locally hosted guess the phrase gameshow system for home use.

Players will be able to connect to the server over the local wifi network and use their phones as input devices for game actions.

I will add more here as I add more into the project. Right now it is just a proof of concept.

Important: I had to allow this traffic through Windows Firewall for the wireless connections to work.

Game reads puzzles as two column CSV files. I create them in Google docs with a headers of Clue | Puzzle. It doesn't matter what the top colmuns say though, but it does need to have at least a blank column cuz the first gets purged. I'll get around to a template sometime.
